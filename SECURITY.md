# Security Policy  
Effective: 2025

Thank you for your interest in helping keep this project safe.  
About-Face Tax Group LLC (AFTG) is committed to protecting users of the Wander Buddy Dashboard and all related digital systems.

## Supported Versions
The following versions are currently supported with security updates:

| Version | Status            |
|--------|--------------------|
| v5.x   | Active / Supported |
| <5.x   | Deprecated         |

Security fixes will only be applied to the latest public version of this repository.

---

## Disclosure Policy

- We respectfully request that you do not publicly disclose vulnerabilities until we have confirmed and addressed them.  
- Coordinated disclosure timelines will be discussed individually.  
- If the issue affects other users or systems, we may request additional time to patch securely.

---

## Scope

This policy applies to:

- Wander Buddy Dashboard (all versions)  
- Any code inside this repository  
- Any hosted versions using GitHub Pages  
- Files or assets distributed as downloadable products

Out of scope:

- Social engineering attempts  
- Physical device security  
- Third-party services or websites not controlled by AFTG
